# INTERIER

## Описание

**INTERIER** — учебный проект интернет-магазина мебели. Сайт позволяет просматривать каталог товаров, фильтровать позиции, добавлять их в корзину и оформлять заказ. На главной странице представлены подборки по комнатам (отдых, работа, кухня, детская, ванная) и специальные предложения.

Основные страницы:

| Файл | Назначение |
|------|------------|
| `index.html` | Главная |
| `catalog.html` | Каталог с фильтрами |
| `cart.html` | Корзина и оформление заказа |
| `landing.html` | Отдельная лендинг-страница (учебный макет) |

## Запуск проекта

Проект — статический сайт, для работы не требуется Node.js или сборщик.

### Вариант 1: открыть в браузере

Откройте файл `index.html` в браузере (двойной щелчок или перетаскивание файла в окно браузера).

### Вариант 2: Live Server (рекомендуется)

В VS Code установите расширение **Live Server**, откройте папку проекта и запустите сервер для `index.html`. В настройках проекта указан порт **5501** (см. `.vscode/settings.json`).

### Редактирование стилей (SCSS)

Стили собраны в `style.scss` и подключены на страницах через скомпилированный `style.css`. При изменении SCSS перекомпилируйте файл, например:

```bash
sass style.scss style.css
```

Для этого нужен установленный [Dart Sass](https://sass-lang.com/install/).

## Используемые технологии

- **HTML5** — разметка страниц
- **CSS3 / SCSS (Sass)** — стили, адаптивная вёрстка (брейкпоинты для планшета и мобильных)
- **JavaScript (Vanilla)** — переключение фильтра в каталоге (`filter-toggle.js`)
- **Google Fonts** — шрифт [Montserrat](https://fonts.google.com/specimen/Montserrat)
- **SVG** — логотип и иконки соцсетей
